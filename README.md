# ⚡ Sai Raghunadh — Futuristic Anti-Gravity Portfolio

A premium, award-winning developer portfolio with a **Dark Neon Cyberpunk** theme.

## 🚀 Quick Start

Simply open `index.html` in any modern browser — no build tools needed.

```
port/
├── index.html          ← Main portfolio page
├── styles.css          ← Full cyberpunk stylesheet
├── script.js           ← Animations, canvas stars, typing effect
├── profile.jpg         ← ⬅ ADD YOUR PHOTO HERE
├── astronaut.png       ← Floating astronaut illustration
├── project_learning.png
├── project_calculator.png
└── project_food.png
```

## 📸 Adding Your Profile Photo

1. Copy your photo to `C:\Users\SAI RAGHUNATH\Downloads\port\`
2. Rename it to **`profile.jpg`**
3. Reload the browser — the photo will replace the "SR" initials automatically

> The photo works best as a square portrait (min 400×400px)

## ✏️ Personalizing Links

Update these placeholders in `index.html` (search for the URL):

| Element | Current Value | Replace With |
|---|---|---|
| GitHub | `https://github.com/` | Your GitHub profile URL |
| LinkedIn | `https://linkedin.com/` | Your LinkedIn profile URL |
| LeetCode | `https://leetcode.com/` | Your LeetCode profile URL |
| Resume | `alert(...)` | Link to your PDF resume |
